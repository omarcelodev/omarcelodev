<h3 align="center">Quem Sou Eu?</h3>

Sou estudante de `Engenharia de Software` no `Centro Universitário de Goiás` (UniGoiás) e minha jornada na tecnologia começou pela curiosidade de entender como sites funcionam. Aprendi `HTML, CSS e JavaScript` com o Curso em Vídeo do Gustavo Guanabara e fui me aprofundando até me apaixonar por `backend, dados e IA.`

Meus primeiros projetos surgiram logo no primeiro período da faculdade: um `sistema de lançamento de notas em C e um modelo de LLM em Python para prever os valores do milho` — projetos simples, mas que definiram meu caminho.
Hoje estou consolidando minha base em desenvolvimento `fullstack`, trabalhando em um sistema web de agendamentos para barbearias com `Java, Spring Boot, PostgreSQL e React`. É um projeto real que une tudo que venho aprendendo.
Meu objetivo é me especializar em `Engenharia de Dados e Machine Learning`, áreas que combinam o que mais me interessa: resolver problemas complexos com tecnologia.

Gosto de trabalhar em equipe com fluxo organizado no estilo `Kanban`, e no ambiente acadêmico costumo assumir o papel de `gestor dos projetos em grupo` — um perfil de `liderança` que faz parte de quem `eu sou`.
